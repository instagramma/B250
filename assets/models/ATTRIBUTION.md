# Lab 2 · 3D Anatomy Explorer — Model Attributions

The 3D models in this folder are stored **locally** with the app (never hotlinked), so external
outages / deleted URLs / CORS can't break the viewer.

**Every model here MUST have a redistributable license** (CC0 / CC BY / CC BY-SA or equivalent). Do not
add anything with unclear, non-commercial, or missing redistribution terms. For each model record:
source, version, creator, license, the exact required attribution text, and any modifications made.

The viewer reads attribution from `models.json` (per-model `attribution` field) and displays it in the
Explorer. Keep this file and `models.json` in sync.
